<h1 align="center">pixi-grid-examples</h1>

# Instructions

## Install

```sh
git clone https://github.com/armathai/pixi-grid-examples
cd pixi-grid-examples
npm install
```

## Usage

### to start a dev server

```sh
npm start
```

### to build single html file in dist folder

```sh
npm run build
```
